# MyBlog
我的个人博客

# 我的网址
http://120.24.93.68

# 开发框架
后端springboot+jpa+thymeleaf

# 部署
1.将sql的url和密码换成你的

2.将文件打包，并将jar包传到服务器

3.java -jar jar包名 --server.port=指定端口

提示：如果部署失败，可能是没有创建数据库
